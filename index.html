Aqui você encontrará alguns jogos que desenvolvi utilizando HTML, CSS e JavaScript. Para começar a jogar, basta clicar na imagem do jogo desejado. Ao terminar, clique no logo do site para voltar à página inicial.

Corrida do Encanador:

Experimente a emoção de um clássico com um toque moderno! Nesse jogo, um encanador está correndo por uma fase repleta de canos. Sua missão é simples: pular os canos para evitar bater e continuar avançando.


Como Jogar?

No celular:

Toque na tela para fazer o encanador pular.

No computador:

Aperte qualquer tecla do teclado para saltar.


Conforme você avança, a fase acelera, tornando o jogo cada vez mais desafiador. Teste suas habilidades e veja até onde você consegue ir!
